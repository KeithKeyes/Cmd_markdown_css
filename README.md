# Cmd 作业部落 编辑器 自定义格式
## 1.项目说明
 为作业部落(Cmd markdown)的自定义样式设计的css.
 此css主要在helloqingfeng的设计基础上进行了一些修改,使得文中代码更易阅读.
特别提醒:建议设置为暗色背景,这样看起来更cool哦!

## 3.使用方法
 
 复制*Cmd markdown css确定版*文件中的内容, 粘贴至Cmd作业部落中的**自定义样式**中的**自定义css**中.
